AI Job Recommendation System (MCP Tool)

This project is an AI-driven job recommendation platform that analyzes a user’s resume and maps it to real-time job opportunities while providing actionable career guidance.

Key Features

Resume Intelligence

Extracts skills, experience, and domain knowledge

Generates resume summary

Identifies skill gaps and missing areas

Creates a future learning roadmap and preparation strategy

Job Discovery

Fetches live job data from LinkedIn Jobs and Naukri

Uses keyword-based and skill-based matching

Recommends relevant roles aligned with the candidate’s profile

Explainable AI

LLM explains why a job is suitable

Highlights gaps to improve employability
